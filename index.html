FAQ

Apa itu DitzMarketplace?

DitzMarketplace adalah layanan API gratis yang menyediakan berbagai endpoint untuk mempermudah pengembang mengakses data dan fitur canggih, seperti AI, integrasi pihak ketiga, dan lainnya.

Apa saja endpoint yang ada?

Kami menyediakan berbagai endpoint, termasuk Downloader API (YouTube, TikTok, Instagram, dll.), Search API, Anime & Manga API, dan Random Image API.

Apakah DitzMarketplace benar-benar gratis?

Ya, DitzMarketplace sepenuhnya gratis dan tidak memiliki batasan kuota harian atau bulanan.

Apakah data saya aman saat menggunakan DitzMarketplace?

Kami berkomitmen untuk menjaga keamanan data pengguna. Silakan baca kebijakan privasi kami untuk informasi lebih lanjut.

Apakah saya bisa menyarankan fitur baru?

Tentu! Kami sangat terbuka untuk saran dan masukan. Anda bisa bergabung dengan komunitas kami untuk memberikan saran.

Apakah DitzMarketplace memiliki batasan penggunaan?

Tidak ada batasan penggunaan. Anda dapat mengakses semua API kami kapan saja sesuai kebutuhan.
